# Business Analytics Dashboard (2020–2024)

Ce projet présente un tableau de bord Power BI complet permettant d’analyser les inscriptions, les performances et les profils utilisateurs d’une entreprise fictive entre 2020 et 2024.  
Il illustre un cas d’usage professionnel : préparation des données, modélisation, création de KPIs et visualisation des insights clés.

---

## 📊 Objectifs du projet

- Analyser l’évolution des inscriptions et du chiffre d’affaires  
- Identifier les activités les plus populaires  
- Comprendre les profils des inscrits  
- Étudier la saisonnalité  
- Évaluer les performances des centres  
- Fournir un support d’aide à la décision
